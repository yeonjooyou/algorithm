# 알고리즘 문제풀이

> 내가 직접 푼 알고리즘 문제풀이 소스코드 저장소



<hr>


## 소개

> 온라인 저지(OJ :Online Judge)에서 푼 문제들에 대한 풀이 코드 저장소입니다. 
>
> 알고리즘 능력 향상을 위해 1일 1문제 풀이를 목표로 합니다.
>
> 아래 구성과 같은 사이트에서 다양한 언어로 풀이를 합니다. 
>
> 풀이에 주력으로 사용하는 언어는 **Python**입니다. 



## 구성

> 사이트>언어>코드 구조로 디렉터리를 구성합니다.
